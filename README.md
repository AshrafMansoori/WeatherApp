# 🌤️ Weather App

A simple and responsive Weather App built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather data from the **OpenWeather API** and displays current weather information for any city.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays current temperature
* 🤗 Shows "Feels Like" temperature
* 💧 Displays humidity
* 🌬️ Wind speed information
* 📊 Atmospheric pressure
* 🌤️ Weather condition (Clouds, Clear, Rain, etc.)
* 📅 Displays the current date
* 💾 Saves the last searched city using Local Storage
* 📱 Responsive and clean UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 🔑 API Key

This project uses the OpenWeather API.

1. Create a free account at https://openweathermap.org/
2. Generate your API key.
3. Replace the API key in `script.js` with your own.

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```

> **Note:** For learning purposes, the API key is used on the client side. In production, API keys should be stored securely on a backend server.
